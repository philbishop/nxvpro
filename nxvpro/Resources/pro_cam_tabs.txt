<b>Device
Shows a list of the selected camera's properties

Use this view to change the camera's display name
Use this view to select or create a camera group

<b>Storage
View videos captured from the selected camera

Local (NX-V) - videos captured by NX-V
FTP - configure and view remote videos
Onboard - search videos
<i>Not all devices support onboard storage

<b>Location
View or set the selected camera's location

<b>Users
Create, edit or delete users
<i>If supported by the selected camera

<b>System
Shows a list of the camera/device's capabilities
Shows the latest system log from the camera
<i>If supported by the selected camera
